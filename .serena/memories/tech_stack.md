# Technology Stack & Architecture

## Structure
- **Contracts**: Solidity (Foundry) in `contract/`
- **Frontend**: Next.js 16 (App Router), React 19, Tailwind 4, Biome in `frontend/`
- **Scripts**: TypeScript (`tsx`, `viem`, `jose`) in `scripts/`
- **Docs**: `.kiro/steering/` (Source of Truth)

## Key Technologies
- **Protocol**: Uniswap v4 (Hooks), Yellow Network (State Channels), Circle/Arc (USDC/CCTP)
- **Dev Tools**: Foundry, Biome, Bun (Frontend)

## Development Standards
- English thinking, Japanese output for docs.
- 3-phase workflow: Requirements -> Design -> Tasks -> Implementation.
- Conventional Commits.
