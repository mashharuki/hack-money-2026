# Project Overview: Zombie L2 Clearinghouse

## Vision
A financial layer that tokenizes computational resources of underutilized Ethereum L2 chains as **CPT (Compute Token)** and generates USDC revenue via gasless arbitrage.

## Core Value
- **For L2 Operators**: Monetize idle capacity as a stable revenue source (USDC).
- **For Traders**: Arbitrage gas price volatility.
- **For Developers**: Future cost hedging.

## Key Components
1.  **Compute Token (CPT)**: ERC20 representing gas units (GSU).
2.  **Uniswap v4 Market**: CPT/USDC pool with **Hooks** that adjust fees based on L2 utilization.
3.  **Yellow Network**: Gasless, high-speed intent-based arbitrage.
4.  **Arc + Circle**: Settlement of arbitrage profits into USDC managed in an **Operator Vault**.

## Target Chains
- Base Sepolia
- WorldCoin Sepolia
